# Changelog

All notable changes to this project will be documented in this file.

The format is inspired by [Keep a Changelog](https://keepachangelog.com/), and this project follows semantic versioning during active development.

## [0.1.0] - Initial public version

### Added

- Core Lumina plugin system.
- TypeScript-first architecture.
- Plugin lifecycle support.
- ThemePlugin with light/dark mode support.
- DropdownPlugin with declarative triggers and menus.
- PasswordVisibilityPlugin for password field toggling.
- ShimmerPlugin for update feedback effects.
- SpotlightPlugin for cursor-following glow effects.
- CascadePlugin for staggered reveal animations.
- ToastPlugin for temporary notifications.
- TooltipPlugin for contextual tooltips.
- SwitchPlugin for animated panel switching.
- TransitionPlugin for page/navigation transitions.
- ConstellationPlugin for interactive particle backgrounds.
- StormPlugin for animated electric canvas effects.
- AccessibilityPlugin with:
  - color blindness filters,
  - reduced motion,
  - inverted colors,
  - high contrast,
  - focus highlight,
  - large font mode,
  - dyslexia-friendly mode.
- Interactive LuminaFX showcase site.
- OpenDyslexic font support through CSS variables.
- Ambient background toggle between Constellation and Storm.

### Changed

- Reworked the original playground into a project showcase site.
- Improved plugin demos and documentation examples.

### Removed

- Removed the experimental IconPlugin from the active plugin set.
